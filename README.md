# 🌸 Dear Saniya - A Premium Friendship Website

A beautiful, premium-quality, fully responsive website dedicated to a dear friend Saniya. This is a caring friendship website created to bring a smile to her face, reduce stress, and remind her that someone genuinely wishes the best for her.

## Features

✨ **10 Beautiful Pages:**
1. **Home** - Animated welcome with typing effect
2. **Smile Gallery** - 100+ wholesome GIFs
3. **Shayari Collection** - Poetry and quotes in Hindi
4. **Music Therapy** - Curated healing sounds

6. **Happiness Challenge** - 30-day daily tasks
7. **Memory Garden** - Flowers with hidden messages
8. **Care Corner** - Wellness tools & exercises
9. **Dream Board** - Goal inspiration
10. **Final Gift** - Cinematic ending message
